Heroku Cheat Sheet 2017-01-17

Command
$ heroku login login to heroku cli
$ git push heroku master push master to heroku
$ heroku ps:scale web=1 make sure one dyno running
$ heroku ps check state of app dynos
$ heroku open open app in web browser
$ heroku open open app in web browser
$ heroku logs view logs locally
$ heroku logs --tail
Rails specific
$ heroku run rake db:migrate migrate rails db on heroku
Postgres specific
$ heroku pg:push [LOCAL DB NAME] [HEROKU DB NAME] --app [HEROKU APP NAME] push local db to heroku instance
