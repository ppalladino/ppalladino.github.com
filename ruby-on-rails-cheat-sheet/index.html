RoR Cheat Sheet 2016-12-09

Notes from guides.rubyonrails.org

Naming Convention
controllers are plural Ex: Users
models are singular Ex: User
Debugging
install byebug $ gem install byebug
set breakpoint by including byebug
General
start new project $ rails new [PROJECT]
start dev server $ bin/rails server
run tests $ bin/rails test
$ bin/rails test:models
$ bin/rails test TEST=[PATH TO TEST]
shows routes $ bin/rails routes
Database
generate general migration $ bin/rails generate migration name_of_some_migration
generate migration with auto constructed script $ bin/rails generate migration [DESCRIPTION]to[TABLE] [FIELD]:[TYPE]
drops and then recreates the database and includes your seeds.rb $ rake db:reset
execute migrations $ bin/rails db:migrate
$ bin/rails db:migrate RAILS_ENV=production
execute seeds $ bin/rails db:seed
set local environment $ bin/rails server â€“environment production
Generators
generate controller $ bin/rails generate controller [CONTROLLER PLURAL]
generate model $ bin/rails generate model [MODEL SINGULAR] [PROP]:string [PROP]:text
Testing
generate integration test $ bin/rails generate integration_test [TEST NAME]
